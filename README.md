## Viewing the deck

1. Double-click `deck/index.html` — it opens in your browser.
2. Arrow keys advance / reverse slides.
3. Press **`s`** to open presenter view (speaker notes + timer).
4. Press **`Esc`** for slide overview.
5. Press **`?`** for the full keyboard shortcut list.

Tested in Chrome and Safari. No install required — the reveal.js library
loads from a CDN, so you'll need to be online the first time you open it.
